       - LlamaGuard step: if cfg.llama_guard.enabled → result = _llama_guard_scan(text, cfg.llama_guard) (использует **non-normalized** text — LlamaGuard видит оригинальный input для семантики); if result → return.

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-02-02 | DoS | LlamaGuard hung / slow | mitigate | httpx.Client timeout=cfg.timeout_ms (full connect+read+write); fail-open on TimeoutException |
